# Tictactoe

A simple react app to play tictactoe in your browser. Made by following this [tutorial](https://react.dev/learn/tutorial-tic-tac-toe).

# How to use
```
git clone https://github.com/B1LLP4RK/TicTacToe.git
cd TicTacToe
npm start
```
